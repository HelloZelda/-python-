# 口算天天练

## 简介
这是一个简单的口算练习和测试程序，支持加法和减法练习，并可以查看历史记录。

## 功能
1. **练习模式**: 练习加法或减法。
2. **测试模式**: 测试加法或减法，并显示正确率。
3. **历史记录**: 查看之前的测试结果。

## 使用方法

### 运行程序
```bash
python 口算天天练_ver1.0.py
```

### 选择模式
1. 输入 `1` 进入练习模式。
2. 输入 `2` 进入测试模式。
3. 输入 `3` 查看历史记录。

### 练习和测试模式
1. 选择加法或减法（输入 `1` 或 `2`）。
2. 输入题目数量。
3. 输入最大加数或被减数。
4. 根据提示回答问题，程序会给出正确与否的反馈。

### 历史记录模式
- 查看之前测试的结果记录。

## 注意事项
- 确保在运行程序前创建 `sampleList.txt` 文件，以便查看历史记录功能正常工作。
